    func CreateOrder ¶ 

    func CreateOrder(c *gin.Context)

    CreateOrder accepts the create order request and stores order details in the DB.

    func FetchOrder ¶ 

    func FetchOrder(c *gin.Context)

    FetchOrder retrieves the order details from DB based on the 'order_id' passed in path param.

    func InitDB ¶ 

    func InitDB()

    InitDB establishes a connection to the required DB (based on SQL connection string). Initializes the DB with default tables required for running Order Microservice.

    func Ping ¶ 

    func Ping(c *gin.Context)

    Ping is used to check the health of the ORDER service. If service is up and running, it returns a status of '200 OK'

    func UpdateOrder ¶ 

    func UpdateOrder(c *gin.Context)

    UpdateOrder retrieves an order details from DB based on the 'order_id' passed in path param. After fetching order details, it updates the old details with new details for the same order record.
